Al enviar una petición POST a la ruta http://localhost:3000/registro con un formato JSON que contiene el nombre "Maria" y el mensaje "Hola comunidad", el servidor procesa los datos de manera exitosa.

La respuesta exacta devuelta por el servidor es:
{"estado":"Datos recibidos","nombre":"Maria","mensaje":"Hola comunidad"}

Esto demuestra que el middleware express.json() configurado en server.js funciona perfectamente, permitiendo al servidor leer los datos entrantes en el cuerpo de la petición (req.body) y responder con un objeto JSON en base a ellos.
